# Package index

## Layer Functions

Functions that allow you to add layers to your plot, such as lanes and
markers.

- [`geom_swim_lane()`](https://chop-cgtinformatics.github.io/ggswim/reference/geom_swim_lane.md)
  : Add lanes to swimmer plots
- [`geom_swim_marker()`](https://chop-cgtinformatics.github.io/ggswim/reference/geom_swim_marker.md)
  : Add markers to swimmer plots
- [`geom_swim_arrow()`](https://chop-cgtinformatics.github.io/ggswim/reference/geom_swim_arrow.md)
  : Add arrows to swimmer plot lanes

## Scale Functions

Functions that assist with cleaning up the legend and assigning
aesthetic connections.

- [`scale_arrow_discrete()`](https://chop-cgtinformatics.github.io/ggswim/reference/scale_arrow_discrete.md)
  : Position scales for discrete arrow data
- [`scale_marker_discrete()`](https://chop-cgtinformatics.github.io/ggswim/reference/scale_marker_discrete.md)
  : Position scales for discrete marker data

## Aesthetic and Theme Functions

Functions to use icons, change the theme, and make your swimmer plot
pop!

- [`fontawesome()`](https://chop-cgtinformatics.github.io/ggswim/reference/fontawesome.md)
  : Retrieve FontAwesome Unicode
- [`search_fontawesome()`](https://chop-cgtinformatics.github.io/ggswim/reference/search_fontawesome.md)
  : Search for FontAwesome aliases to include in ggswim
- [`bootstrap()`](https://chop-cgtinformatics.github.io/ggswim/reference/bootstrap.md)
  : Retrieve Bootstrap Unicode
- [`load_fonts()`](https://chop-cgtinformatics.github.io/ggswim/reference/load_fonts.md)
  : Load Select Fonts
- [`search_bootstrap()`](https://chop-cgtinformatics.github.io/ggswim/reference/search_bootstrap.md)
  : Search for Bootstrap aliases to include in ggswim
- [`theme_ggswim()`](https://chop-cgtinformatics.github.io/ggswim/reference/theme_ggswim.md)
  [`theme_ggswim_dark()`](https://chop-cgtinformatics.github.io/ggswim/reference/theme_ggswim.md)
  : Apply custom theme styling for ggplot2 plots
